## نیازمندی‌ها

- macOS 14.0 (سونوما) یا جدیدتر
- حساب کاربری با دسترسی مدیر (Administrator)
- اتصال شبکه (وای‌فای یا اترنت)

## نصب

### دانلود (پیشنهادی)

1. به صفحه [Releases](https://github.com/TronIsHere/dns-switcher/releases/latest) بروید.
2. فایل **DNS-Switcher-macOS.zip** را دانلود کنید.
3. روی فایل zip دوبار کلیک کنید تا **DNS Switcher.app** استخراج شود.
4. برنامه را به پوشه **Programs** (Applications) بکشید.

### اولین باز کردن (Gatekeeper)

چون این برنامه از خارج Mac App Store توزیع می‌شود، ممکن است macOS اولین بار هشدار امنیتی نشان دهد.

**روش الف:** روی **DNS Switcher.app** راست‌کلیک کنید → **Open** (باز کردن) → در پنجره **Open** را دوباره بزنید.

**روش ب:** **System Settings** → **Privacy & Security** → پیام برنامه مسدودشده → **Open Anyway**.

## اولین اجرا

در اولین اجرا صفحه خوش‌آمدگویی نمایش داده می‌شود:

1. **زبان** را انتخاب کنید — انگلیسی یا فارسی.
2. خلاصه قابلیت‌ها را ببینید.
3. روی **شروع کنید** بزنید.

بعداً می‌توانید زبان را از صفحه اصلی تغییر دهید.

## نمای کلی صفحه اصلی

| بخش | کاربرد |
|-----|--------|
| **شبکه** | انتخاب اینترفیس (وای‌فای، اترنت و …) |
| **DNS فعلی** | سرورهای DNS در حال استفاده |
| **پیش‌فرض DNS** | گزینه‌های سریع: پیش‌فرض، کلودفلر، گوگل، کواد۹، سفارشی |
| **پروفایل‌های ذخیره‌شده** | تنظیمات نام‌گذاری‌شده برای دسترسی سریع |
| **اعمال DNS** | اعمال پیش‌فرض، پروفایل یا آدرس‌های سفارشی |
| **بازگشت به خودکار** | بازگرداندن DNS به حالت DHCP |

نوار وضعیت پایین صفحه موفقیت یا خطا را نشان می‌دهد.

## انتخاب شبکه

macOS می‌تواند چند سرویس شبکه داشته باشد. DNS Switcher همه سرویس‌های فعال را فهرست می‌کند.

1. منوی **سرویس شبکه** را باز کنید.
2. اتصالی که استفاده می‌کنید را انتخاب کنید (معمولاً **Wi-Fi** یا **Ethernet**).
3. کارت **DNS فعلی** به‌روز می‌شود.

تغییر DNS فقط روی سرویس انتخاب‌شده اعمال می‌شود. اگر هم وای‌فای و هم اترنت دارید، قبل از اعمال، سرویس درست را انتخاب کنید.

## پیش‌فرض‌های DNS

| پیش‌فرض | سرورها | مناسب برای |
|---------|--------|------------|
| **پیش‌فرض** | خودکار (DHCP) | DNS روتر یا ISP |
| **کلودفلر** | `1.1.1.1`، `1.0.0.1` | سرعت و حریم خصوصی |
| **گوگل** | `8.8.8.8`، `8.8.4.4` | DNS عمومی پایدار |
| **کواد۹** | `9.9.9.9`، `149.112.112.112` | امنیت و مسدودسازی بدافزار |
| **سفارشی** | آدرس‌های خودتان | DNS پیشرفته یا محلی |

روی کارت پیش‌فرض کلیک کنید، سپس **اعمال DNS** را بزنید.

انتخاب **پیش‌فرض** و اعمال، DNS را خودکار می‌کند (مثل **بازگشت به خودکار**).

## DNS سفارشی

1. پیش‌فرض **سفارشی** را انتخاب کنید.
2. آدرس **اصلی** را وارد کنید (الزامی).
3. در صورت نیاز آدرس **ثانویه** را وارد کنید.
4. **اعمال DNS** را بزنید.

فرمت‌های پشتیبانی‌شده:

- IPv4 (مثلاً `1.1.1.1`)
- IPv6 (آدرس‌های دارای `:`)

آدرس نامعتبر تا اصلاح نشود اعمال نمی‌شود.

## پروفایل‌های ذخیره‌شده

با پروفایل می‌توانید نام و سرورهای DNS را برای تعویض سریع ذخیره کنید.

### ساخت پروفایل

1. DNS را تنظیم کنید (پیش‌فرض یا سفارشی).
2. **ذخیره به‌عنوان پروفایل** (یا **افزودن پروفایل**) را بزنید.
3. نامی وارد کنید (مثلاً «کار»، «خانه»، «بازی»).
4. **ذخیره پروفایل** را بزنید.

### استفاده از پروفایل

روی کارت پروفایل کلیک کنید، سپس **اعمال DNS**.

### ویرایش یا حذف

از کارت پروفایل ویرایشگر را باز کنید، سرورها یا نام را تغییر دهید، یا پروفایل را حذف کنید.

## اعمال و بازنشانی DNS

### اعمال DNS

1. سرویس شبکه و پیش‌فرض یا پروفایل را انتخاب کنید.
2. **اعمال DNS** را بزنید.
3. رمز مدیر macOS را وارد کنید.
4. منتظر پیام موفقیت و به‌روز شدن **DNS فعلی** بمانید.

### بازگشت به خودکار

**بازگشت به خودکار** DNS دستی را پاک می‌کند و DHCP را برمی‌گرداند. رمز مدیر لازم است.

## تغییر زبان

از کنترل زبان در بالای صفحه استفاده کنید:

- **English** (انگلیسی)
- **فارسی** — چیدمان راست‌به‌چپ (RTL) فعال می‌شود

زبان انتخابی ذخیره می‌شود و در اجرای بعدی بازگردانده می‌شود.

## رمز مدیر سیستم

DNS Switcher تنظیمات شبکه سیستم را تغییر می‌دهد. macOS برای هر اعمال یا بازنشانی، تأیید مدیر می‌خواهد.

- پنجره رمز، دیالوگ استاندارد macOS است؛ رمز داخل برنامه وارد نمی‌شود.
- با لغو پنجره، DNS تغییر نمی‌کند.
- DNS Switcher رمز شما را ذخیره نمی‌کند.

## عیب‌یابی

### خطای «Command failed» یا مشکل دسترسی

- مطمئن شوید حساب شما Administrator است.
- رمز درست را در پنجره macOS وارد کنید.
- بعد از باز کردن System Settings دوباره امتحان کنید.

### DNS تغییر نکرد

- **سرویس شبکه** درست را انتخاب کنید (وای‌فای در برابر اترنت).
- بعد از تغییر پیش‌فرض، **اعمال DNS** را بزنید.
- در **Terminal** دستور `scutil --dns` را اجرا کنید (پیشرفته).

### برنامه باز نمی‌شود

- اولین بار راست‌کلیک → **Open**.
- **System Settings** → **Privacy & Security** را بررسی کنید.

### DNS فعلی «خودکار» است ولی DNS سفارشی گذاشتم

DNS خالی یا از DHCP ممکن است **خودکار** نشان داده شود. پیش‌فرض را دوباره روی سرویس درست اعمال کنید.
